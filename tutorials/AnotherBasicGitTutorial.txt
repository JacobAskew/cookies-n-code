Step 6: Initialize a Git Repository
Type "git init" and press Enter. This tells Git that you want to start tracking changes in this folder.

Step 7: Add Your Script to Git
Type "git add your_script.py" (replace "your_script.py" with the actual name of your Python script).
Press Enter. This tells Git to start watching your script for changes.

Step 8: Commit Your Changes
Type "git commit -m 'Initial commit'" and press Enter. This saves a snapshot of your script as it is right now.

Go back to your GitHub repository's page and copy the "HTTPS" link (you'll find it under the green "Code" button).

Step 10: Link Your Local Repository to GitHub
In the terminal, type "git remote add origin https://github.com/your-username/your-repository.git" (replace "your-username" with your GitHub username and "your-repository" with your repository name).
Press Enter. This connects your local repository to the one on GitHub.

Step 11: Push Your Code to GitHub
Type "git push -u origin main" and press Enter. This sends your code to GitHub.
